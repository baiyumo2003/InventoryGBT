
# Project Overview

This directory contains the code, data processing pipeline, and training scripts for our language modeling experiments using **Llama** and **GPT-2** with the Hugging Face ecosystem.

## Repository Structure

```text
.
├── dataset/
├── GPT2/
├── Llama3.1/
└── readme.md
````

## Folder Description

### `dataset/`

This folder contains the scripts used to process the raw data.


### `GPT2/`

This folder provides the training scripts and related code for fine-tuning or training **GPT-2** using the **Hugging Face** framework.


### `Llama3.1/`

This folder provides the training scripts and related code for fine-tuning or training **LLaMA 3.1** using the **Hugging Face** structure.
